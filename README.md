# cv_scoring

## Project Description

cv_scoring is a full-stack application designed to score and evaluate CVs (Resumes) using AI-powered analysis. The project combines modern web technologies to provide automated CV evaluation capabilities.

## Technology Stack

- **TypeScript (50.3%)** - Primary language for frontend/backend type-safe development
- **Python (48.8%)** - Backend processing and AI/ML scoring algorithms
- **Supporting Technologies** - HTML, JavaScript, Mako templates, and Docker containerization

## Project Structure

The repository contains a main CVMatch-AI module, suggesting the core functionality is an AI-powered matching and scoring system for CVs.

## Key Features

- **CV Analysis & Scoring** - Automated evaluation of resumes/CVs
- **AI-Powered Matching** - Likely matches CVs with job requirements or criteria
- **Full-Stack Architecture** - Backend API (Python) with frontend interface (TypeScript)
- **Containerized Deployment** - Docker support for easy deployment

## Use Cases

- HR/Recruitment automation
- Resume screening and ranking
- Job-candidate matching
- CV quality assessment
